## Project No: 0012
## Project Name: MyProject

**Objective**
This project focuses on understanding **App-level Templates, Static Files, and URL Routing using include()**.

---
# Step 1 — Create Project and App

```bash
django-admin startproject MyProject
cd MyProject
python manage.py startapp App1
```

---
# Step 2 — Register App
Open:
```
MyProject/settings.py
```

Add app name in `INSTALLED_APPS`:

```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'App1',
]
```

---
# Step 3 — Create App-level Templates and Static
Create the following folders:

```
App1/
│
├── templates/
│   └── App1/
│       └── static_example.html
│
└── static/
    └── App1/
        └── images/
            └── my_pic.png
```

---
# Step 4 — Configure settings.py
Open:

```
MyProject/settings.py
```

### Ensure Templates Configuration
```python
import os

TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [],
        'APP_DIRS': True,
        'OPTIONS': {
            'context_processors': [
                'django.template.context_processors.debug',
                'django.template.context_processors.request',
                'django.contrib.auth.context_processors.auth',
                'django.contrib.messages.context_processors.messages',
            ],
        },
    },
]
```

---
### Static Configuration
```python
STATIC_URL = '/static/'
STATICFILES_DIRS = []
```

---
# Key Points

- `APP_DIRS = True` → Enables app-level templates
- Templates path → `App1/templates/App1/`
- Static path → `App1/static/App1/`
- No need for `STATICFILES_DIRS` for app-level static

---
# Step 5 — Create Template
```
App1/templates/App1/static_example.html
```

```html
{% load static %}

<!DOCTYPE html>
<html>
<head>
    <title>Static Example</title>
</head>
<body>

    <h1>Static File Demo</h1>

    <img src="{% static 'App1/images/my_pic.png' %}" 
         alt="Image not found" width="300">

</body>
</html>
```

---
# Step 6 — Create View
Open:
```
App1/views.py
```

```python
from django.shortcuts import render

def static_example(request):
    return render(request, 'App1/static_example.html')
```

---
# Step 7 — Create App-level URLs
Create:
```
App1/urls.py
```

```python
from django.urls import path
from . import views

urlpatterns = [
    path('static-demo/', views.static_example, name='static_example'),
]
```

---
# Step 8 — Configure Project-level URLs
Open:
```
MyProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('App1/', include('App1.urls')),
]
```

---
# Step 9 — Run Server
```bash
python manage.py runserver
```

http://127.0.0.1:8000/App1/static-demo/
