## Project No: 0001
## Project Name: WelcomeProject

**Objective**
This project focuses on understanding **Django Project Level Configuration (No App Level)**.

# Step 1 — Create Django Project
```bash
django-admin startproject WelcomeProject
```

### Project Structure
After creating the project, the structure will look like this:
```
WelcomeProject/
│
├── manage.py
│
└── WelcomeProject/
    ├── __init__.py
    ├── settings.py
    ├── urls.py
    ├── asgi.py
    └── wsgi.py
```

# Step 2 — Enter Into the Project Folder
```bash
cd WelcomeProject
```

# Step 3 — Create Templates and Static Folders
Create the following folders at **project level**:
```
WelcomeProject/
│
├── templates/
└── static/
```

### Modify `settings.py`
File location:
```
WelcomeProject/WelcomeProject/settings.py
```

### Configure Templates Directory
```python
import os
'DIRS': [os.path.join(BASE_DIR, 'templates')],
```

### Configure Static Files Directory
Add the following line at the bottom of `settings.py`:
```python
STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]
```

# Step 4 — Add Image and HTML File
### Static Folder
```
static/
└── logo.png
```

### Templates Folder
```
templates/
└── t1.html
```

### HTML Content (`t1.html`)
```html
<b>Hello, {{ name }}</b>
```

# Display Image in HTML
## Method 1 — Using Static Tag (Recommended)
```html
{% load static %}
<img src="{% static 'logo.png' %}" alt="Image not found">
```

## Method 2 — Using Static Path
```html
<img src="/static/logo.png" alt="Image not found">
```

# Step 5 — Define View Function
Create a file:
```
WelcomeProject/views.py
```

## Process 1 — Using `HttpResponse`
```python
from django.http import HttpResponse

def m1(request):
    return HttpResponse("Hello, John")
```

## Process 2 — Using `render`
```python
from django.shortcuts import render

def m1(request):
    return render(request, 't1.html', {'name': 'John'})
```

# Step 6 — Define URL Pattern
Open:
```
WelcomeProject/urls.py
```

Add the following code:
```python
from django.contrib import admin
from django.urls import path
from .views import m1

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', m1),
    # path('extraurl/', m1)
]
```

# Step 7 — Start Development Server
```bash
python manage.py runserver
```

# Step 8 — Send HTTP Request
Open browser and visit:
```
http://127.0.0.1:8000/
```
