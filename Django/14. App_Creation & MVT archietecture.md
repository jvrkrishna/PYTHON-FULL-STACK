## Project No: 0005
## Project Name: FirstAppProject

**Objective**
This project focuses on understanding **Django App Creation and MVT Architecture (Model-View-Template)**.

---
# Step 1 — Create Django Project
```bash
django-admin startproject FirstAppProject
```

---
# Step 2 — Enter Into the Project Folder
```bash
cd FirstAppProject
```

---
# Step 3 — Create Django App
```bash
python manage.py startapp app1
```

---
# Project Structure
```
FirstAppProject/
│
├── manage.py
│
├── app1/
│   ├── __init__.py
│   ├── admin.py
│   ├── apps.py
│   ├── models.py
│   ├── views.py
│   ├── tests.py
│   └── migrations/
│
└── FirstAppProject/
    ├── __init__.py
    ├── settings.py
    ├── urls.py
    ├── asgi.py
    └── wsgi.py
```

---
# Step 4 — Register App
Open:
```
FirstAppProject/settings.py
```
Add app name in `INSTALLED_APPS`:
```python
INSTALLED_APPS = [
    'app1',
]
```

---
# Step 5 — Create View in App
Open:
```
app1/views.py
```

```python
from django.http import HttpResponse

def home(request):
    return HttpResponse("Welcome to Django App")
```

---
# Step 6 — Configure URL Mapping
Open:
```
FirstAppProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path
from app1.views import home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', home),
]
```

---
# Step 7 — Start Development Server
```bash
python manage.py runserver
```

---
# Step 8 — Open in Browser
```
http://127.0.0.1:8000/
```

