## Project No: 0014
## Project Name: BootstrapProject

**Objective**
This project focuses on creating a **Professional Menu Bar using Bootstrap (Navbar + Dropdown)** with Django.

---
# Step 1 — Create Project and App
```bash
django-admin startproject BootstrapProject
cd BootstrapProject

python manage.py startapp app1
```

---
# Step 2 — Register App
📄 BootstrapProject/settings.py
```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.staticfiles',
    'app1',
]
```

---
# Step 3 — Create Templates Folder
```bash
mkdir templates
mkdir static
mkdir media
```

---
# Step 4 — Configure Templates
```python
TEMPLATES = [
    {
        'DIRS': [BASE_DIR / "templates"],
        'APP_DIRS': True,
    },

```

---
# Step 5 — Configure Static & Media
```python
STATIC_URL = '/static/'

MEDIA_URL = '/media/'

```

---
# Step 6 — Create Base Template (Navbar)
📄 templates/base.html

```html
{% load static %}
<!DOCTYPE html>
<html>
<head>
    <title>{% block title %}My Project{% endblock %}</title>

    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

    <!-- Bootstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

</head>
<body>

<!-- Navbar -->
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <div class="container-fluid">

        <a class="navbar-brand" href="{% url 'home' %}">MyProject</a>

        <button class="navbar-toggler" type="button" data-bs-toggle="collapse"
                data-bs-target="#navbarNav">
            <span class="navbar-toggler-icon"></span>
        </button>

        <div class="collapse navbar-collapse" id="navbarNav">

            <ul class="navbar-nav">

                <li class="nav-item">
                    <a class="nav-link" href="{% url 'app1:index' %}">Index</a>
                </li>

                <li class="nav-item">
                    <a class="nav-link" href="{% url 'app1:about' %}">About</a>
                </li>

                <li class="nav-item">
                    <a class="nav-link" href="{% url 'app1:contact' %}">Contact</a>
                </li>

                <!-- Dropdown -->
                <li class="nav-item dropdown">
                    <a class="nav-link dropdown-toggle" data-bs-toggle="dropdown">
                        More
                    </a>

                    <ul class="dropdown-menu">
                        <li><a class="dropdown-item" href="{% url 'app1:index' %}">Index</a></li>
                        <li><a class="dropdown-item" href="{% url 'app1:about' %}">About</a></li>
                        <li><a class="dropdown-item" href="{% url 'app1:contact' %}">Contact</a></li>
                        <li><a class="dropdown-item" href="{% url 'app1:home' %}">App Home</a></li>
                    </ul>
                </li>

            </ul>

        </div>
    </div>
</nav>

<!-- Content -->
<div class="container mt-4">
    {% block content %}{% endblock %}
</div>

</body>
</html>
```

# Step 9 — App Views
📄 app1/views.py

```python
from django.shortcuts import render

def index(request):
    return render(request, 'app1/index.html')

def about(request):
    return render(request, 'app1/about.html')

def contact(request):
    return render(request, 'app1/contact.html')

def home(request):
    return render(request, 'app1/home.html')
```

# Step 11 — App URLs
📄 app1/urls.py

```python
from django.urls import path
from . import views

app_name = 'app1'

urlpatterns = [
    path('', views.index, name='index'),
    path('about/', views.about, name='about'),
    path('contact/', views.contact, name='contact'),
    path('home/', views.home, name='home'),
]
```

---
# Step 12 — Project URLs
📄 BootstrapProject/urls.py

```python
from django.contrib import admin
from django.urls import path, include
from .views import home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', home, name='home'),
    path('app1/', include('app1.urls')),
]
```
