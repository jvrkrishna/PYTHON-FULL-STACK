## Project No: 0013 (Day 3)
## Topic: Static + Media + Final Navigation

**Objective**
- Add CSS using Static files
- Display Images using Media files
- Complete navigation system

---
# Step 1 — Create Static Folder
📁 Structure:
```
App1/
    static/
        css/
            style.css
```

---
📄 App1/static/css/style.css
```css
body {
    background-color: #f0f8ff;
    text-align: center;
}

h1 {
    color: darkblue;
}
```

---
# Step 2 — Create Media Folder
📁 Structure:
```
App1/
    media/
        sample.jpg
```
👉 Place any image as `sample.jpg`

---
# Step 3 — Configure Settings
📄 Sample_Project/settings.py
```python
import os
STATIC_URL = '/static/'
MEDIA_URL = '/media/'
MEDIA_ROOT = BASE_DIR / "App1/media"
```

---
# Step 4 — Create Template
📄 App1/templates/App1/App1_home.html
```html
{% load static %}
<!DOCTYPE html>
<html>
<head>
    <title>App1 Home</title>

    <!-- CSS -->
    <link rel="stylesheet" href="{% static 'css/style.css' %}">
</head>
<body>
<h1>App1 Home Page</h1>
<!-- Image -->
<img src="{{ MEDIA_URL }}sample.jpg" width="300">
<nav>
    <a href="{% url 'home' %}">Home</a> |
    <a href="{% url 'App1_index' %}">Index</a> |
    <a href="{% url 'App1_about' %}">About</a> |
    <a href="{% url 'App1_contact' %}">Contact</a>
</nav>

</body>
</html>
```

---
# Step 5 — Create View
📄 App1/views.py
```python
def App1_home(request):
    return render(request, 'App1/App1_home.html')
```

---
# Step 6 — Update App URLs
📄 App1/urls.py
```python
path('home/', views.App1_home, name='App1_home'),
```

---
# Step 7 — Serve Media Files
📄 Sample_Project/urls.py
```python
from django.conf import settings
from django.conf.urls.static import static

urlpatterns += static(settings.MEDIA_URL, document_root=settings.MEDIA_ROOT)
```

---
# Step 8 — Run Server
```bash
python manage.py runserver
```
