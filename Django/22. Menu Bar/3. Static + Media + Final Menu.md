## Project No: 0013 (Day 3)
## Topic: Static (CSS) + Media (Image)

**Objective**
- Apply CSS styling
- Display image

---
# Step 1 — Create Static Folder
📁 Structure:
```
App1/
    static/
        css/
            style.css
```

---
📄 App1/static/css/style.css
```css
body {
    background-color: #f0f8ff;
    text-align: center;
}

h1 {
    color: darkblue;
}

nav a {
    margin: 10px;
    text-decoration: none;
    color: green;
}
```

---
# Step 2 — Create Media Folder
📁 Structure:
```
App1/
    media/
        sample.jpg
```

---
# Step 3 — Configure Settings
📄 Sample_Project/settings.py
```python
STATIC_URL = '/static/'

MEDIA_URL = '/media/'
MEDIA_ROOT = BASE_DIR / "App1/media"
```

---
# Step 4 — Update Existing Template (Example: index.html)
📄 App1/templates/App1/index.html
```html
{% load static %}
<!DOCTYPE html>
<html>
<head>
    <title>Index Page</title>
    <!-- CSS -->
    <link rel="stylesheet" href="{% static 'css/style.css' %}">
</head>
<body>
<h1>Index Page</h1>

<!-- Image -->
<img src="{{ MEDIA_URL }}sample.jpg" width="300">

<nav>
    <a href="{% url 'home' %}">Home</a> |
    <a href="{% url 'App1_about' %}">About</a> |
    <a href="{% url 'App1_contact' %}">Contact</a>
</nav>

</body>
</html>
```

---
# Step 5 — Serve Media Files
📄 Sample_Project/urls.py
```python
from django.conf import settings
from django.conf.urls.static import static

urlpatterns += static(settings.MEDIA_URL, document_root=settings.MEDIA_ROOT)
```

---
# Step 6 — Run Server
```bash
python manage.py runserver
```
