## Project No: 0013 (Day 1)
## Topic: Project Level Menu (Basic Navigation)

---
# Step 1 — Create Project
```bash
mkdir 0013
cd 0013
django-admin startproject Sample_Project
cd Sample_Project
python manage.py startapp App1
```

---
# Step 2 — Register App
📄 Sample_Project/settings.py
```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.staticfiles',
    'App1',
]
```

---
# Step 3 — Create Templates Folder
```bash
mkdir templates
```

---
# Step 4 — Configure Templates
```python
TEMPLATES = [
    {
        'DIRS': [BASE_DIR / "templates"],
        'APP_DIRS': True,
    },
]
```

---
# Step 5 — Create Home Page
📄 templates/home.html
```html
<h1>Welcome to Project Home Page</h1>

<nav>
    <a href="">Home</a> |
    <a href="">Index</a> |
    <a href="">About</a> |
    <a href="">Contact</a>
</nav>
```

---
# Step 6 — Create View
📄 Sample_Project/views.py
```python
from django.shortcuts import render

def home(request):
    return render(request, 'home.html')
```

---
# Step 7 — URL Mapping
📄 Sample_Project/urls.py

```python
from django.contrib import admin
from django.urls import path
from .views import home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', home),
]
```

---
# Step 8 — Run Server
```bash
python manage.py runserver
```
