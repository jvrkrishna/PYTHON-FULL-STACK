## Project No: 0013 (Day 2)
## Topic: App Level Templates + Navigation

**Objective**
- Create app-level pages (Index, About, Contact)
- Connect navigation between pages
- Link project-level and app-level templates

---
# Step 1 — Create App Templates Folder
📁 Structure:
```
App1/
    templates/
        App1/
            index.html
            about.html
            contact.html
```

---
# Step 2 — Create index.html
📄 App1/templates/App1/index.html
```html
<h1>Index Page</h1>
<nav>
    <a href="{% url 'home' %}">Home</a> |
    <a href="{% url 'App1_about' %}">About</a> |
    <a href="{% url 'App1_contact' %}">Contact</a>
</nav>
```

# Step 5 — Create Views (App Level)
📄 App1/views.py
```python
from django.shortcuts import render
def App1_index(request):
    return render(request, 'App1/index.html')

def App1_about(request):
    return render(request, 'App1/about.html')

def App1_contact(request):
    return render(request, 'App1/contact.html')
```

---
# Step 6 — Create App URLs
📄 App1/urls.py
```python
from django.urls import path
from . import views
urlpatterns = [
    path('', views.App1_index, name='App1_index'),
    path('about/', views.App1_about, name='App1_about'),
    path('contact/', views.App1_contact, name='App1_contact'),
]
```

---
# Step 7 — Connect App URLs to Project
📄 Sample_Project/urls.py
```python
from django.contrib import admin
from django.urls import path, include
from .views import home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', home, name='home'),
    path('App1/', include('App1.urls')),
]
```
