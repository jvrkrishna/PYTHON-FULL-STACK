## Project No: 0008
## Project Name: MultiAppProject

**Objective**
This project focuses on understanding **Multiple Apps in a Single Django Project**.

---
# Step 1 — Create Project
```bash
django-admin startproject MultiAppProject
cd MultiAppProject
```

---
# Step 2 — Create Multiple Apps
```bash
python manage.py startapp app1
python manage.py startapp app2
```

---
# Step 3 — Register Apps
Open:
```
MultiAppProject/settings.py
```

```python
INSTALLED_APPS = [
    'app1',
    'app2',
]
```

---
# Step 4 — Create Views
## app1/views.py
```python
from django.http import HttpResponse

def app1_home(request):
    return HttpResponse("This is App1")
```

---
## app2/views.py
```python
from django.http import HttpResponse

def app2_home(request):
    return HttpResponse("This is App2")
```

---
# Step 5 — Configure URL Mapping
Open:
```
MultiAppProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path
from app1.views import app1_home
from app2.views import app2_home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('app1/', app1_home),
    path('app2/', app2_home),
]
```

---
# Step 6 — Run Server
```bash
python manage.py runserver
```

---
# Output
```
http://127.0.0.1:8000/app1/ → This is App1
http://127.0.0.1:8000/app2/ → This is App2
```
