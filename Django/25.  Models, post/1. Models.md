# What is a Model?
A **Model** is a Python class that represents a **database table**.

- Each attribute → column  
- Each object → row

## Database Table

## ORM

# Validation in Django Models

👉 Django models can be used for validation, but they are not primarily designed for user-facing validation like forms.

In Django, models include basic and optional validation mechanisms.
---

## ✅ What Models Can Validate
### 1. Field-level validation
- `max_length`, `blank=False`, `null=False`  
- Built-in validators like `EmailField`, `URLField`  

---
### 2. Custom validators
from django.core.exceptions import ValidationError

def validate_age(value):
    if value < 18:
        raise ValidationError("Age must be 18+")

---
### 3. Model-wide validation using clean()
from django.db import models
from django.core.exceptions import ValidationError

class Student(models.Model):
    age = models.IntegerField()

    def clean(self):
        if self.age < 18:
            raise ValidationError("Too young")

---
### 4. full_clean() method
- Runs all validations (field + clean)  
- ⚠️ Not called automatically when saving  
student.full_clean()  
student.save()  

---
## ⚠️ Important Limitation
👉 Models do NOT automatically validate on .save()

# Step 4 — Create Model
📄 blog/models.py

```python
from django.db import models
from datetime import datetime

class Post(models.Model):

    title = models.CharField(max_length=150)
    description = models.TextField()
    author = models.CharField(max_length=50)
    created_at = models.DateTimeField(default=datetime.now)
    is_published = models.BooleanField(default=False)

    def __str__(self):
        return self.title
```

# Step 7 — Register Model in Admin
📄 blog/admin.py
```python
from django.contrib import admin
from .models import Post

class PostAdmin(admin.ModelAdmin):

    list_display = ('id', 'title', 'author', 'created_at')
    list_display_links = ('title',)
    list_filter = ('author', 'created_at')

admin.site.register(Post, PostAdmin)
```
