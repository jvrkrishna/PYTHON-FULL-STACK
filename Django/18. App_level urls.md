## Project No: 0009
## Project Name: AppURLProject

**Objective**
This project focuses on understanding **App Level URL Configuration using include()**.

---
# Step 1 — Create Project & App
```bash
django-admin startproject AppURLProject
cd AppURLProject
python manage.py startapp app1
```

---
# Step 2 — Register App
```python
INSTALLED_APPS = ['app1']
```

---
# Step 3 — Create App URLs File
Create:
```
app1/urls.py
```

```python
from django.urls import path
from .views import home

urlpatterns = [
    path('', home),
]
```

---
# Step 4 — Create View
```python
from django.http import HttpResponse

def home(request):
    return HttpResponse("App Level URL")
```

---
# Step 5 — Configure Project URLs
Open:
```
AppURLProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('app1/', include('app1.urls')),
]
```

---
# Output
```
http://127.0.0.1:8000/app1/ → App Level URL
```
