## Project No: 0004
## Project Name: WelcomeProject

**Objective**

This project focuses on understanding **Static Files in Django (Images, CSS, JS)** at project level.

---
# Step 1 — Create Django Project
```bash
django-admin startproject WelcomeProject
```

---
# Step 2 — Enter Into the Project Folder

```bash
cd WelcomeProject
```

---
# Step 3 — Create Required Folders
Create the following folders at project level:
```
WelcomeProject/
│
├── templates/
└── static/
```

---
# Step 4 — Configure Templates in settings.py
Open:
```
WelcomeProject/WelcomeProject/settings.py
```
Modify `DIRS`:
```python
import os

'DIRS': [os.path.join(BASE_DIR, 'templates')],
```

---
# Step 5 — Configure Static Files
Add the following line in `settings.py`:
```python
STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]
```

---
# Step 6 — Add Files
### Static Folder
```
static/
└── logo.png
```

### Templates Folder
```
templates/
└── t1.html
```

---
# Step 7 — HTML File with Static Image
```html
{% load static %}
<h2>Hello, {{ name }}</h2>
<img src="{% static 'logo.png' %}" alt="Image not found">
```

---
# Step 8 — Create View
```
WelcomeProject/views.py
```

```python
from django.shortcuts import render
def m1(request):
    return render(request, 't1.html', {'name': 'John'})
```

---
# Step 9 — Configure URL Mapping
Open:
```
WelcomeProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path
from .views import m1
urlpatterns = [
    path('admin/', admin.site.urls),
    path('', m1),
]
```

---
# Step 10 — Start Development Server
```bash
python manage.py runserver
```

---
# Step 11 — Open in Browser
```
http://127.0.0.1:8000/
```

---
# Output
- Hello, John
- Image (logo.png) displayed on page
