## Project No: 0007
## Project Name: FirstAppProject

**Objective**
This project focuses on understanding **Static Files (CSS, Images, JS) inside App with Templates**.

---
# Step 1 — Create Django Project
```bash
django-admin startproject FirstAppProject
```

---
# Step 2 — Enter Into the Project Folder
```bash
cd FirstAppProject
```

---
# Step 3 — Create Django App
```bash
python manage.py startapp app1
```

---
# Step 4 — Register App
Open:
```
FirstAppProject/settings.py
```

Add app name:
```python
INSTALLED_APPS = [
    'app1',
]
```

---
# Step 5 — Create Templates Folder (Inside App)
```
app1/
└── templates/
    └── t1.html
```

---
# Step 6 — Create Static Folder (Inside App)
```
app1/
└── static/
    ├── style.css
    └── logo.png
```

---
# Step 7 — Write CSS File
Create:
```
app1/static/style.css
```

```css
h2 {
    color: blue;
    text-align: center;
}
```

---
# Step 8 — Create HTML File
```
app1/templates/t1.html
```

```html
{% load static %}

<h2>Hello, {{ name }}</h2>

<link rel="stylesheet" href="{% static 'style.css' %}">

<img src="{% static 'logo.png' %}" alt="Image not found">
```

---
# Step 9 — Create View
Open:
```
app1/views.py
```

```python
from django.shortcuts import render

def home(request):
    return render(request, 't1.html', {'name': 'John'})
```

---
# Step 10 — Configure URL Mapping
Open:
```
FirstAppProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path
from app1.views import home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', home),
]
```

---
# Step 11 — Start Development Server
```bash
python manage.py runserver
```

---
# Step 12 — Open in Browser
```
http://127.0.0.1:8000/
```

---
# Output
- Hello, John (styled with CSS)
- Image displayed (logo.png)

---
# Static File Flow
```
Template → {% load static %} → static files → Browser
```

- App-level Static Files
- CSS Integration in Django
- Image Display using static
