## Project No: 0011
## Project Name: DataPassProject

**Objective**
This project focuses on understanding **Passing Data from View to Template**.

---
# Step 1 — Create Project & App
```bash
django-admin startproject DataPassProject
cd DataPassProject
python manage.py startapp app1
```

---
# Step 2 — Register App
📄 DataPassProject/settings.py
```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.staticfiles',
    'app1',
]
```

---
# Step 3 — Create Templates Folder
📁 Create path:
```
app1/templates/
```

---
# Step 4 — Create Template
📄 app1/templates/t1.html
```html
<h2>Name: {{ name }}</h2>
<h2>Age: {{ age }}</h2>
```

---
# Step 5 — Create View (App Level)
📄 app1/views.py
```python
from django.shortcuts import render
def home(request):
    data = {
        'name': 'John',
        'age': 25
    }

    return render(request, 't1.html', data)
```

---
# Step 6 — Create App URLs
📄 app1/urls.py  (create file)
```python
from django.urls import path
from .views import home

urlpatterns = [
    path('', home),
]
```

---
# Step 7 — Connect App URLs to Project
📄 DataPassProject/urls.py
```python
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('app1.urls')),
]
```

---
# Step 8 — Run Server
```bash
python manage.py runserver
```

---
# Output
```
Name: John
Age: 25
```

---
# Flow
```
View → Data(Dictionary) → Template → Output
```

---
# How It Works
## In View
```python
data = {'name': 'John'}
```

## In Template
```html
{{ name }}
```

👉 Django replaces variable with value
