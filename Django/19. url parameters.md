## Project No: 0010
## Project Name: URLParamProject

**Objective**
This project focuses on understanding **Capturing Parameters from URL**.

---
# Step 1 — Create Project & App
```bash
django-admin startproject URLParamProject
cd URLParamProject
python manage.py startapp app1
```

---
# Step 2 — Create View
```python
from django.http import HttpResponse

def greet(request, name):
    return HttpResponse(f"Hello, {name}")
```

---
# Step 3 — Configure URL
```python
from django.urls import path
from .views import greet

urlpatterns = [
    path('hello/<str:name>/', greet),
]
```

---
# Output
```
http://127.0.0.1:8000/hello/John/ → Hello, John
```

---
# Concepts Covered
- URL Parameters
- Dynamic URLs
- Path converters (`str`, `int`)
