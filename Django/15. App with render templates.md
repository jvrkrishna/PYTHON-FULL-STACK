## Project No: 0006
## Project Name: FirstAppProject

**Objective**
This project focuses on understanding **Templates with App using render() (MVT Architecture in action)**.

---
# Step 1 — Create Django Project
```bash
django-admin startproject FirstAppProject
```

---
# Step 2 — Enter Into the Project Folder
```bash
cd FirstAppProject
```

---
# Step 3 — Create Django App
```bash
python manage.py startapp app1
```

---
# Step 4 — Register App
Open:
```
FirstAppProject/settings.py
```
Add app name:
```python
INSTALLED_APPS = [
    'app1',
]
```

---
# Step 5 — Create Templates Folder (Inside App)
```
app1/
└── templates/
    └── t1.html
```

---
# Step 6 — Create HTML File
```html
<h2>Hello, {{ name }}</h2>
```

---
# Step 7 — Create View
Open:
```
app1/views.py
```

```python
from django.shortcuts import render

def home(request):
    return render(request, 't1.html', {'name': 'John'})
```

---
# Step 8 — Configure URL Mapping
Open:

```
FirstAppProject/urls.py
```

```python
from django.contrib import admin
from django.urls import path
from app1.views import home

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', home),
]
```

---
# Step 9 — Start Development Server
```bash
python manage.py runserver
```

---
# Step 10 — Open in Browser
```
http://127.0.0.1:8000/
```

---
# Output
```
Hello, John
```

---
# MVT Flow
```
User → URL → View → Template → Response
```

---
# Concepts Covered
- App-level Templates
- render() with Templates
- Passing Data to HTML
- Template Variables (`{{ name }}`)
- MVT Architecture (Real Flow)
