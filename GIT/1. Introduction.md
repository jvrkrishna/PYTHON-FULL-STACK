1️⃣ What is Git
Git is a Version Control System (VCS) used to track changes in code.

It helps developers:
Track code changes
See who made changes
Work together on the same project
Restore older versions of code
👨‍💻 Created by Linus Torvalds (2005).

2️⃣ What is GitHub
GitHub is a cloud platform that hosts Git repositories.

Think like this:
Tool	   Purpose
Git	   Tracks code locally
GitHub	Stores code online

Benefits of GitHub:
Backup of code
Collaboration with other developers
Project management
Open source sharing
Owned by Microsoft since 2018.

3️⃣ Basic Git Workflow (Very Important)
The Git workflow always follows this order:
Working Directory → Staging Area → Commit → GitHub

Meaning:
Edit file
   ↓
Stage file
   ↓
Commit changes
   ↓
Push to GitHub

4️⃣ Key Git Concepts (Simple Notes)
Repository (Repo)
A project folder tracked by Git.

Example:
my-project/
Clone
Copy a GitHub repository to your computer.
git clone repo_url
Stage
Select files that you want to save in the next commit.

git add filename
or
git add .

Commit
Save a snapshot of staged files.
git commit -m "message"

Example:
git commit -m "Added login page"
Branch
A separate version of the project.

Example:
main
feature-login
feature-payment

Create branch:
git branch branch-name
Merge
Combine branches.

Example:
git merge feature-login
Pull
Get latest changes from GitHub.
git pull origin main
Push
Upload your commits to GitHub.
git push origin main

5️⃣ Installing Git (First Time Setup)
Step 1
Download Git
👉 https://git-scm.com

Step 2
Check installation in VS Code terminal
git --version

Example output:
git version 2.44.0

Step 3 (Important)
Configure your Git identity
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
Check configuration:
git config --list

6️⃣ Create First Git Project in VS Code
Step 1
Open folder in VS Code
File → Open Folder

Step 2
Open terminal
Shortcut:
Ctrl + `

Step 3
Initialize Git
git init
This creates hidden folder:
.git
Now your folder is a Git repository.

7️⃣ First Commit (Important)
Step 1
Create file
index.html

Step 2
Check status
git status
Output:
Untracked files:
index.html

Step 3
Stage file
git add index.html
or
git add .

Step 4
Commit
git commit -m "First commit"
Now Git saved your project snapshot.

Step 2

Step 3
Enter
Repository name
Description
Public / Private

9️⃣ Connect Local Project to GitHub
Copy repository URL.
Example:
https://github.com/username/project.git
Add remote
git remote add origin https://github.com/username/project.git
Check remote:
git remote -v

🔟 Push Code to GitHub
Push code:
git push -u origin main
Now your code is uploaded to GitHub 🎉

11️⃣ Daily Git Commands (Very Important)

12️⃣ Git History
See commit history:
git log

Short version:
git log --oneline

Example:
a45bc2 First commit
b87df4 Added login page

13️⃣ Undo Changes
Undo file changes:
git checkout filename

Undo staging:
git reset filename

Undo commit:
git reset --soft HEAD~1
