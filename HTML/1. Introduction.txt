Definition
A website is a collection of related web pages that are stored on a web server and accessed through the Internet using a web browser. All the pages in a website are usually connected by links and share a common domain name.

Example
www.example.com
www.google.com
Uses of a Website
Providing information (news, education)
Entertainment (videos, games)
Communication (email, social media)
Online shopping (e-commerce)
Business promotion

Key Components of a Website
Web Pages
Individual documents of a website
Written using HTML
Styled using CSS
Interactive using JavaScript
Domain Name
The address of the website

Example: www.example.com
Web Server
A computer that stores website files
Sends pages to users when requested

Web Hosting
Service that provides space on a server to store website files
Content Management System (CMS) (Optional)
Software used to create and manage websites easily
Examples: WordPress, Joomla, Drupal

ğŸ“„ 4. Difference Between Static and Dynamic Web Pages
Feature	Static Web Page	Dynamic Web Page
Content	Same for all users	Changes for users
Technologies	HTML, CSS	HTML, CSS, JS + Backend
Database	Not required	Required
Speed	Faster	Slightly slower
User Interaction	Very limited	Highly interactive
Examples	Portfolio, brochure site	Social media, e-commerce

Simple Explanation
Static Page: Ready-made content (like a printed poster)
Dynamic Page: Content generated on demand (like a live TV show)
